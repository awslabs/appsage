## Features
-Accepts a single string parameter (`message`).
- Returns the same message with an `echo2:` prefix.
- Helpful as a debugging tool to confirm that your server is functioning end-to-end.

---

## Usage Guide

### Parameters
- **message** (string, required)  
  The text you want to send to the Echo tool.

---

### Example Input
```
json
{
  "message": "Hello World"
}
```
